# SMART VOICE RECEPTION AI

نظام استقبال مكالمات ذكي بالذكاء الاصطناعي للمطاعم.

يستقبل المكالمات بصوت احترافي، يفهم اللهجة المحلية، يقرأ مشاعر الزبائن، ويأخذ طلباتهم — 24 ساعة بلا توقف.

## المميزات

- 🎙️ **صوت بشري أصيل** — لهجة فلسطينية حقيقية
- 😊 **قراءة المشاعر** — يحلل نبرة الصوت ويعرف مزاج الزبون
- 🏷️ **خصومات ذكية** — يعطي خصومات تلقائياً حسب المزاج
- 🌍 **متعدد اللغات** — عربي، إنجليزي، عبري
- 🛡️ **تشغيل 24/7** — لا تفوت أي مكالمة
- ⚡ **رد فوري** — يأخذ الطلب في أقل من 3 دقائق

## الصفحات

| الصفحة | المحتوى |
|--------|---------|
| 🏠 **الرئيسية** | Hero + مميزات + إحصائيات + CTA |
| 📞 **المكالمة** | محاكاة مكالمة حية + محادثة AI + تحليل مشاعر |
| 📋 **الطلبات** | قائمة + فلتر + تفاصيل + تغيير الحالة |
| 📊 **لوحة التحكم** | بطاقات إحصائية + رسوم بيانية |
| 📈 **الإحصائيات** | دونات تشارت + خط زمني + أداء AI |

## التقنيات

- React 18 (via CDN + Babel)
- Tajawal + Cairo fonts (Google Fonts)
- عربي RTL كامل
- وضع داكن / فاتح
- بدون build step — افتح `index.html` مباشرة

## التشغيل محلياً

```bash
# افتح الملف مباشرة
open index.html

# أو عبر سيرفر محلي
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```
